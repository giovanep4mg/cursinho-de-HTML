  1. O que o elemento < iframe > é usado para fazer em HTML?
    • a) Adicionar links internos
    • b) Incorporar conteúdo externo

  1. Como tornar um campo de formulário obrigatório em HTML?

  1. Qual é o atributo usado para desativar a interação com um elemento de formulário?

  1. O que o elemento < details > é usado para criar em HTML?

  1. Qual tag é usada para abreviaturas em HTML?

    • c) < input type="text" uneditable >

  1. O que a tag < summary > representa em conjunto com < details > em HTML?
