Aqui estão alguns aspectos adicionais sobre HTML que ainda não foram abordados:

Campos de Formulário e Atributo placeholder:

Em campos de formulário, você pode usar o atributo placeholder para fornecer uma breve dica ou exemplo do tipo de entrada esperado.

Elementos de Controle de Formulário:

Além da tag < input >, existem outros elementos de controle de formulário, como < select > para menus suspensos,
e < textarea > para áreas de texto multilineares.

Listas de Definição:

Além de listas ordenadas e não ordenadas, HTML oferece a tag < dl > para listas de definição, que consistem em pares de termos (dt) e suas definições (dd).

Tags < b > e < i > :

Embora não seja considerado as melhores práticas para formatação moderna, as tags < b > (negrito) e < i > (itálico) ainda existem e são utilizadas para estilizar texto.

A tag < hgroup > é usada para agrupar vários elementos de cabeçalho (< h1 > a < h6 > quando eles pertencem a uma única seção ou tópico.

Atributo colspan em Tabelas:

O atributo colspan é utilizado nas células de uma tabela para mesclar várias colunas em uma única célula.
